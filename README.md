## Instalación

```shell
  $ git clone https://github.com/Farmalive/daemon-a2f-node.git
  $ cd daemon-a2f-node
  $ npm install
  $ mkdir logs
  $ cd lib && mkdir config

```

## Ejecución

```shell
  $ npm start
```

## Pruebas

```shell
  $ npm test
```
